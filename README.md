
# Cyclistic Bike-Share Case Study

## Project Overview
This project analyzes Cyclistic bike-share usage patterns between annual members and casual riders using SQL, Google BigQuery, and Excel dashboards.

The objective was to identify behavioral differences between rider types and provide marketing recommendations to convert casual riders into annual members.

---

## Tools Used
- Google BigQuery
- SQL
- Microsoft Excel
- GitHub

---

## Data Cleaning Process
- Prepared 2019 and 2020 datasets
- Standardized inconsistent column names
- Combined datasets using UNION ALL
- Removed invalid ride durations
- Standardized rider categories

---

## Analysis Performed
- Total ride analysis
- Average ride duration analysis
- Weekday ride frequency analysis
- Ride duration trends by weekday
- Total riding hours comparison

---

## Key Insights
- Members ride significantly more frequently
- Casual riders take rides over 3x longer
- Casual riders prefer weekends
- Members show weekday commuting behavior

---

## Business Recommendations
- Launch weekend-focused membership promotions
- Promote long-ride membership savings
- Target recreational riders for membership conversion
- Encourage weekday riding habits
