
# Cyclistic Bike-Share Case Study

## Project Overview
This project analyzes Cyclistic bike-share usage patterns between annual members and casual riders using SQL, BigQuery, and Excel.

## Tools Used
- Google BigQuery
- SQL
- Microsoft Excel

## Business Task
Analyze how annual members and casual riders use Cyclistic bikes differently and provide marketing recommendations to convert casual riders into members.

## Data Cleaning Process
- Combined 2019 and 2020 datasets
- Standardized rider categories
- Removed invalid ride durations
- Created cleaned analytical views

## Key Insights
- Members ride more frequently
- Casual riders ride over 3x longer
- Casual riders prefer weekends
- Members show weekday commuting behavior

## Recommendations
- Launch weekend-focused membership campaigns
- Promote long-ride membership savings
- Target recreational riders for conversion

## Dashboard
The dashboard was created in Excel using exported BigQuery analysis results.
